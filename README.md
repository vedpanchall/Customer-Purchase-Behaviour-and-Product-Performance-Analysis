Customer Purchase Behaviour & Product Performance Analysis Dashboard

Project Overview

-> This project analyzes customer purchasing behavior and product performance to uncover insights related to revenue,
customer segments, product popularity, seasonal trends, and subscription impact. The dashboard is designed to support data-driven business decisions through interactive visuals and KPIs.

Key KPIs

-> Total Customers
-> Total Purchase Amount (Revenue)
-> Average Purchase Amount
-> Average Review Rating

Key Insights

-> Clothing category contributes the highest revenue
-> Certain age groups show higher average spending
-> Seasonal trends highlight peak sales periods
-> Subscribers show different purchasing behavior compared to non-subscribers
-> Some products have high sales but lower ratings, indicating improvement opportunities

Tools & Technologies

-> Power BI
-> Data Modeling & DAX
-> Data Cleaning & Transformation
-> Interactive Dashboard Design
